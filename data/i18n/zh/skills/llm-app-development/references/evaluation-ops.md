# 评估与运维

## 日志

记录：

- request id。
- 用户输入摘要。
- prompt 版本。
- 模型和参数。
- 工具调用。
- 检索证据。
- 输出摘要。
- 错误和延迟。

避免记录：

- API key。
- 密码。
- 完整个人敏感信息。
- 未脱敏业务机密。

## 成本控制

- 使用小模型做分类、重写和摘要。
- 长文档先检索后读取。
- 对重复上下文使用缓存。
- 对高成本工具设置限流。

## 评估

最小评估集包括：

- 典型用户任务。
- 边界输入。
- 工具失败。
- 无答案问题。
- 安全和权限问题。

## 发布

上线前检查：

- 配置可通过环境变量注入。
- 外部服务超时和重试已设置。
- 有降级路径。
- 有人工接管或问题上报入口。
